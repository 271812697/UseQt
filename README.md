## opengl

这是一个尝试在Qt下编写的OpenGL物理渲染器

**实现的功能有：**

- 实现了简单的骨骼动画
- 实现了PBR 和IBL
- 使用计算着色器烘培IBL
- 使用计算着色器实现bloom效果
